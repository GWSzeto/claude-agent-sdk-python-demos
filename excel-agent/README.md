# Excel Agent

An AI assistant for creating, editing, and analyzing Excel spreadsheets with professional formatting.

## What It Does

- Create spreadsheets with data and formulas
- Apply professional formatting (colors, fonts, number formats)
- Read and analyze existing Excel files
- Follow financial model color-coding standards

## Techniques Used

- **MCP Tools** - Custom tools for spreadsheet operations using openpyxl
- **Skills** - Domain knowledge for Excel best practices and formatting standards
- **Evaluator-Optimizer** - Quality validation to ensure spreadsheets have no formula errors and proper formatting

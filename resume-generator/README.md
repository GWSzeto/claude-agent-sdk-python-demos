# Resume Generator

Generates professional resumes by researching a person online and creating a formatted Word document.

## What It Does

- Search the web for professional background info
- Fetch LinkedIn profile data
- Analyze and structure resume content
- Generate a .docx resume with proper formatting

## Techniques Used

- **MCP Tools** - Custom tools for web search, profile fetching, and document generation
- **Tool Use** - Orchestrating multiple tools to gather and process information
- **Evaluator-Optimizer** - Quality scoring loop that iteratively improves the resume until it meets quality thresholds

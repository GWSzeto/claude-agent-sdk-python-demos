# Email Triage Skill

## Priority Levels
- P0 - Critical: "urgent", "critical", "production down", from executives
- P1 - High: Customer support, overdue invoices, P0/P1 bug reports
- P2 - Medium: Internal communications, meeting requests
- P3 - Low: Newsletters, marketing, FYI emails

## Workflow
1. Search for unread emails: `is:unread`
2. Identify P0/P1 emails using keyword search
3. Label appropriately
4. Summarize findings
